# 🗓️ DoTheReco AI

AI 기반 위치/동선 최적화 일정 추천 서비스

## 📌 프로젝트 소개

DoTheReco AI는 사용자의 고정 일정과 유동 일정을 기반으로
빈 시간, 이동 거리, 예상 소요 시간, 자연어 요청을 함께 고려하여
하루 일정을 자동 추천하는 AI 일정 추천 서비스입니다.

단순 캘린더 기능을 넘어,
현재 위치와 다음 일정 사이의 이동 동선을 최소화하도록
유동 일정의 장소를 추천하는 기능을 구현했습니다.

기존 일정 앱은 사용자가 직접 시간을 배치해야 하는 경우가 많았고,
이동 동선이나 실제 하루 흐름까지 고려해 추천해주는 기능은 부족하다고 느껴 프로젝트를 시작했습니다.

---

## ✨ 핵심 기능

### 🤖 AI 자연어 일정 추천

사용자의 자연어 요청을 분석하여 추천 전략에 반영합니다.

예시:

* "이동이 적은 순서로 짜줘"
* "오늘 피곤해서 여유롭게 넣어줘"
* "마감 가까운 일정 먼저 추천해줘"

---

### 📍 위치 기반 일정 추천

현재 위치와 다음 고정 일정을 기준으로
이동 동선을 최소화하는 장소를 추천합니다.

예시:

* 건대 일정 이후 → 건대 근처 카페 추천
* 강남 일정 이후 → 강남역 주변 일정 추천

---

### ⏰ 고정 / 유동 일정 혼합 스케줄링

* 고정 일정(Fixed Schedule)
* 유동 일정(Flexible Task)

을 함께 분석하여 자동 타임라인을 생성합니다.

---

## 🤖 AI 추천 방식

자연어 요청과 위치 기반 동선을 함께 고려하여
추천 전략을 동적으로 변경합니다.

* 이동 최소화 기반 장소 추천
* 여유시간(buffer) 기반 일정 조정
* 마감일(D-Day) 우선 일정 배치
* 현재 위치(currentPlace) 기반 동선 연결

자세한 추천 로직은
[AI Recommendation Docs](./docs/ai-recommendation.md)
에서 확인할 수 있습니다.

## 🧩 Technical Challenges

### 현재 위치와 무관한 장소가 추천되던 문제

초기에는 장소명을 그대로 검색어로 사용하여
현재 위치와 관계없는 장소가 추천되는 문제가 있었습니다.

예시:

* "올리브영" → 서울 전체 랜덤 지점 추천

이를 해결하기 위해 현재 위치 기반 지역 키워드 추론 로직을 추가하여:

* 건국대학교 → 건대입구
* 강남 치과 일정 → 강남역

처럼 지역 키워드로 변환 후 검색하도록 개선했습니다.

또한 currentPlace 기반 동선 연결을 통해
이전 일정 위치를 기준으로 다음 유동 일정 장소를 추천하도록 구현했습니다.
